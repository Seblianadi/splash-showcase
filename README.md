# Splash Showcase

Статичная витрина вариантов сплэшей для приложения.

## Что внутри

- `index.html` - основная страница для просмотра.
- `*.mp4` - видео-варианты сплэшей.
- `winter-park-logo.jpeg` - зимний вид для концепции парк-логотип.
- `presentation_notes.md` - короткая шпаргалка для обсуждения.

## Локальный просмотр

Можно открыть `index.html` напрямую в браузере или запустить простой локальный сервер:

```powershell
python -m http.server 5177 --bind 127.0.0.1
```

После этого открыть:

```text
http://127.0.0.1:5177/index.html
```

## Публикация через GitHub Pages

1. Создать новый репозиторий на GitHub.
2. Загрузить в него все файлы из этой папки.
3. Открыть `Settings` -> `Pages`.
4. Выбрать:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Сохранить настройки.

Через несколько минут сайт будет доступен по адресу:

```text
https://USERNAME.github.io/REPOSITORY_NAME/
```
